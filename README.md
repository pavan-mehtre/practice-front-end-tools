# Practice Front End Tools

A minimal React app scaffolded with Vite for practicing front-end tooling.

## Scripts

- `npm run dev` — start development server (default port 5173)
- `npm run build` — build production bundle (output in `dist/`)
- `npm run preview` — preview the production build

## Run locally

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

Feel free to ask me to add tests, linting, or CI.
